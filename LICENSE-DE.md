Die Creators Project Lizenz ist eine freie lizenz für Software um sicher zustellen dass die entwickler der Software nicht vergessen werden und die software nicht für illegale dinge genutzt wird.

## Definition von Freier Software
In diesem Dokument wird wenn wir von Freier Software reden, Software gemeint die Frei unter den unten genannten bedingungen Kopiert und geteilt werden kann, es wird nicht von kostenloser software gesprochen.

Entwickler, die unsere Creators Project-Lizenz verwenden, schützen Ihre Rechte in zwei Schritten: (1) das Urheberrecht an der Software geltend zu machen und (2) Ihnen diese Lizenz anzubieten, die Ihnen die rechtliche Erlaubnis gibt, die Software zu kopieren, zu verteilen und/oder zu modifizieren.

# Bedingungen und Konditionen
## 1.Quellcode
Der Quellcode muss immer verfügbar sein, wenn die Software verteilt wird. Der Quellcode muss in einem Format bereitgestellt werden, das es anderen Entwicklern ermöglicht, die Software zu ändern und zu verbessern.

Beim verteilen einer veränderten Version der Software muss der geänderte Quellcode ebenfalls bereitgestellt werden.

## 2.Kopien
Die Software darf Kopiert und verteilt werden, aber es darf kein Geld für die Software verlangt werden der Original entwickler muss immer genannt werden und ist der einzige der für die Software Geld verlangen darf.

## 3.Veränderungen
Die Software darf verändert werden, aber auch hierbei muss der Original entwickler genannt werden und die Software darf nicht für Geld verkauft werden.

Die Software darf nicht verwendet werden um informationen zu stehlen oder um andere Menschen zu schaden. Auch dürfen gegenstände nicht durch die Software beschädigt werden.

## 4.Entwickler
Die Entwickler der Software sind nicht verantwortlich für Schäden die durch die Software entstehen. Auch sind die Entwickler nicht verantwortlich für Schäden die durch unveränderte Versionen der Software entstehen.
Die Entwickler sind auch nicht verantwortlich für Schäden die durch veränderte Versionen der Software entstehen.

Alle Rechte an der Software liegen bei den Entwicklern der Software.

Alle Entwickler die an der Software mitgearbeitet haben sind in der Liste der Entwickler genannt.

## 5.Sonstiges
Die Software darf nicht für illegale Dinge verwendet werden.

Über Youtube, Twitch, Discord und Co. darf kein Geld mit unfollständigen Tutorials oder Life Hack Videos verdient werden.

Pornografiche Inhalte müssen immer mit einem Warnhinweis versehen werden.

Für weitere Fragen die Entwickler der Software kontaktieren (z.B. über Github, Youtube, Discord, ...).
